# LIO_OS — Jouw AI Business Operating System

Welkom bij LIO_OS! Dit is je complete toolkit om Claude Code in te zetten voor je business.

## Wat zit erin?

| Map | Wat | Voor wie |
|-----|-----|---------|
| `course/` | 8 modules, 40 lessen + bonus skills | Iedereen — start hier |
| `tools/` | 35+ slash commands (skills) | Power users |
| `systems/` | Complete business systemen (ads, content, automation, video) | Gevorderden |
| `gifts/` | Downloadbare prompts, workflows, cheat sheet | Iedereen |
| `prompts/` | 750+ viral hooks, 30 money prompts, workflows | Content creators |
| `templates/` | CLAUDE.md templates, email sequences, proposals | Business owners |
| `claude-md-templates/` | Kant-en-klare CLAUDE.md per business type | Starters |
| `examples/` | Voorbeeld CLAUDE.md (ingevuld) | Starters |
| `mcp-configs/` | MCP server configuraties (Notion, Gmail, Meta Ads, etc.) | Techneuten |
| `agency-agents-main/` | 50+ gespecialiseerde AI agent prompts | Agencies & teams |
| `install.sh` | Mac/Linux installer | Mac/Linux users |
| `install.ps1` | Windows installer | Windows users |

---

## Snel starten

### Mac / Linux
```bash
cd LIO_OS
chmod +x install.sh
./install.sh
```

### Windows (PowerShell)
```powershell
cd LIO_OS
.\install.ps1
```

### Handmatig
1. Open deze map in **Cursor** of **VS Code**
2. Open de terminal
3. Type: `claude` om Claude Code te starten
4. Type: `/lio_os:start` om de course te beginnen
