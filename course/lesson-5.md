### HTML Build Requirements

**File structure:**
- Create a SINGLE `index.html` file with ALL CSS embedded in a `<style>` tag in the `<head>`
- No separate CSS files. No JavaScript frameworks. No npm. One file, opens anywhere.

**Design quality — this must look PROFESSIONAL:**
- Clean, modern design system with good whitespace and visual hierarchy
- System font stack: `-apple-system, BlinkMacSystemFont, 'Segoe UI', Roboto, sans-serif` OR include a Google Font via CDN `<link>` tag (Inter, Poppins, or similar)
- Professional color scheme derived from their niche:
  - Sports/fitness → bold, energetic (deep blues, bright accents, strong contrasts)
  - Wellness/health → calm, clean (soft greens, warm neutrals, gentle gradients)
  - Tech/SaaS → modern, sharp (dark backgrounds, neon accents, or clean whites with blue)
  - Creative/content → vibrant, expressive (gradients, bold typography, dynamic colors)
  - Business/consulting → authoritative, trustworthy (navy, charcoal, gold accents)
  - If unsure, go with a clean dark/light scheme that feels premium
- Mobile-responsive with media queries (`@media (max-width: 768px)`)
- Smooth subtle touches: box shadows, border-radius, hover states on buttons, gradient backgrounds where appropriate
- Good typography hierarchy: large hero heading, clear subheadings, readable body text
- Proper spacing: generous padding and margins, sections that breathe

**Content — DEEPLY personalized, not generic:**
- Their actual business name in the hero
- Their actual services/offerings listed
- Copy written in their voice/tone from CLAUDE.md
- Their actual target audience referenced throughout
- Real, usable copy — NOT placeholder text, NOT lorem ipsum
- If testimonials are relevant: write realistic dummy testimonials from people in their target market (use realistic names and scenarios for their niche)

**Page structure (minimum sections):**
1. **Hero section** — big headline, subheadline, primary CTA button
2. **Problem/solution or about section** — why their audience needs them
3. **Services/features section** — what they offer (cards, grid, or list)
4. **Social proof section** — testimonials, stats, or trust indicators
5. **CTA section** — final call to action with button
